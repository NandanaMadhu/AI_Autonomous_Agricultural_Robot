# 📡 Bluetooth Communication

AgriVision uses the Raspberry Pi's built-in Bluetooth module to transmit disease predictions to a nearby mobile device.

---

# Communication Flow

Disease Prediction

↓

Bluetooth Module

↓

Mobile Device

↓

Display Disease Name

---

# Advantages

- Wireless communication
- Low power consumption
- Easy mobile integration
