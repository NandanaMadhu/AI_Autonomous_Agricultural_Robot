# ⚙️ Software Architecture

The AgriVision software follows a modular architecture where each subsystem performs a dedicated task.

---

# Layers

1. User Layer
2. Application Layer
3. Computer Vision Layer
4. AI Inference Layer
5. Communication Layer
6. Hardware Interface Layer

---

# Main Modules

- Camera Interface
- OpenCV Processing
- TensorFlow Inference
- Navigation Controller
- Bluetooth Manager
- GPIO Controller

---

# Design Principles

- Modular
- Scalable
- Reusable
- Maintainable
