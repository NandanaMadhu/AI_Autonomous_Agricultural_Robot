# 💻 Software Documentation

This directory contains the software architecture and implementation details of the **AgriVision** autonomous agricultural robot.

The software stack combines computer vision, artificial intelligence, embedded programming, and wireless communication to enable autonomous crop monitoring and plant disease detection.
