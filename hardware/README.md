# 🔩 Hardware Documentation

This directory contains all the hardware-related documentation for the **AgriVision** autonomous agricultural robot.

The robot is designed around a **Raspberry Pi 4** and integrates computer vision, artificial intelligence, obstacle detection, and autonomous navigation to monitor crop health in agricultural fields.

# 🏗 Hardware Overview

The AgriVision robot consists of four major hardware subsystems:

- Raspberry Pi 4 (Main Controller)
- Vision System
- Mobility System
- Communication System

These subsystems work together to perform real-time crop monitoring and disease detection.

---

# 📷 Main Hardware Components

- Raspberry Pi 4
- Raspberry Pi Camera Module
- Ultrasonic Sensor
- L298N Motor Driver
- Four DC Gear Motors
- Four-Wheel Robot Chassis
- Battery Pack
- Built-in Bluetooth

---

# 🎯 Hardware Objectives

- Capture crop images
- Detect plant diseases
- Navigate autonomously
- Avoid obstacles
- Communicate results wirelessly
- Support future hardware expansion
