# 🧠 AI Models

This directory is intended to store trained AI models used for plant disease detection.

## Supported Model Formats

- TensorFlow SavedModel
- .keras
- .h5
- TensorFlow Lite (.tflite)

## Current Status

The original trained model is not included in this repository because it was developed as part of an academic project and is no longer available.

Future versions of this repository may include:

- TensorFlow CNN model
- TensorFlow Lite model for Raspberry Pi
- Quantized edge AI models
